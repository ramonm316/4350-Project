PROGRESS OF OUR APP


This is our mock up of the idea for our app. Here we drew out the initial layout and the functionality of the randomizer. This is a rough sketch of an idea of our app. The end product will be better looking.

This is our first step into making the app. We had created the buttons into making our app have some functionality. At this stage the buttons do not have any function to them. This is where we started. At this point there is no functionality with the buttons.

This is the next part of our app. We had created the functionality for the add and delete buttons. At this point the add button allows us to add any subject we want to the list.The delete button currently allows the user to delete the subject from the list when they type the subject into the entry list.

This is where we had added in an edit button to our app. The edit button allows the user to select the subject of their choice and edit that subject. This will allow the user to be able to change the subject or be specific to what the user wants. There was also an icon added to the application in this version. In this version, the subjects are stored and saved for when you go in and out of the app.
